# lineChart
1.利用UIBezierPath曲线自定义一个数据曲线图，通过CAShapeLayer+CABasicAnimation完成绘制动画，可近100%的还原UI设计图，并添加点击响应事件；<br>
2.根据服务端下发数据，动态计算坐标，实时刷新UI<br>
3.实现正数，小数，百分比，负数等多重类型曲线图
